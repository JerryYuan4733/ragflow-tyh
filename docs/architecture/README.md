## 架构概览

```
┌─────────────────────────────────────────────────────────────────┐
│                        客户端层                                  │
│    Web UI (React)  │  Python SDK  │  HTTP API Client           │
└─────────────────────────────────────────────────────────────────┘
                              │
┌─────────────────────────────────────────────────────────────────┐
│                        网关层 (Nginx)                            │
└─────────────────────────────────────────────────────────────────┘
                              │
┌─────────────────────────────────────────────────────────────────┐
│                      应用服务层                                  │
│         API Server (Quart)  │  Task Executor                   │
└─────────────────────────────────────────────────────────────────┘
                              │
┌─────────────────────────────────────────────────────────────────┐
│                      核心引擎层                                  │
│   RAG Engine │ DeepDoc │ Agent Engine │ GraphRAG │ Memory      │
└─────────────────────────────────────────────────────────────────┘
                              │
┌─────────────────────────────────────────────────────────────────┐
│                      模型服务层                                  │
│   LLM │ Embedding │ Rerank │ TTS │ ASR │ OCR (50+ Providers)   │
└─────────────────────────────────────────────────────────────────┘
                              │
┌─────────────────────────────────────────────────────────────────┐
│                        存储层                                    │
│  MySQL/PostgreSQL │ Elasticsearch/Infinity │ MinIO/S3 │ Redis  │
└─────────────────────────────────────────────────────────────────┘
```

## 技术栈

| 层次 | 技术选型 |
|------|----------|
| **前端** | React, TypeScript, UmiJS, Ant Design |
| **后端** | Python 3.10+, Quart (Async), Peewee ORM |
| **文档引擎** | Elasticsearch / Infinity / OceanBase |
| **关系数据库** | MySQL / PostgreSQL / OceanBase |
| **对象存储** | MinIO / AWS S3 / Azure Blob / OSS |
| **缓存/队列** | Redis |
| **容器化** | Docker, Docker Compose, Kubernetes (Helm) |
